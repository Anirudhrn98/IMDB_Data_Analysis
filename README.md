# IMDB_Data_Analysis

This repository contains implementation details of the coursework at Rochester Institute of Technology (RIT). 

## Course Details 

Course Name : Introduction to Big Data (CSCI 620)

Instructor Name : Micheal Mior

# 

The repository contains implementation details regarding the following topics of Data Analysis. The dataset used for this coursework is the IMDB Dataset which can be referenced by using the following link 

`https://www.imdb.com/interfaces`

1. Designing Relational Model
2. Creating Database
3. Data Cleaning and Normalization
4. Data Integration to MongoDB from PostGresSQL
5. Itemset Mining
6. Data Clustering 
7. Distributed Data Processing
